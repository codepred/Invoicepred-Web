## Installation


**Install the gem:**

```bash
npm install
npm install vite@latest --save-dev
npm install vue-router@4
npm install @vuepic/vue-datepicker
```


**Run for production:**


```bash
npm run build
```

**Run for localhost:**

```bash
npm run dev
```
